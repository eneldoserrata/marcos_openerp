El contenido de esta documentación está dividida en dos partes **Comercial Partners** y **Developers Partners**:

Commercial partners:
--------------------
Este manual explica todo lo relacionado a la parte comercial, desde el primer contacto con el cliente hasta su etapa final.

-------------------
Este Manual incluye la tutoría técnica para que los programadores puedan instalar el sistema Openerp adecuadamente en cualquier plataforma además de conocer su estructura para la creacion y modificacion de modulos.
